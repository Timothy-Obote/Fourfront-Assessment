This project recreates a mobile-first membership cards interface based on a provided design image. It uses HTML, CSS, JavaScript, and Bootstrap 5 to achieve a pixel-perfect, responsive layout with interactive features.

The folder structure includes an index.html file, a css folder with style.css, a js folder with script.js, and an images folder for profile pictures. The HTML structures the page with a white outer container, a menu toggle, several dark grey boxes containing a Michael profile section with golden border, a 2x2 grid of colored member boxes (pink, green, yellow, pink) each with a profile image and name with points, a motivational text box with golden and white text, two separate dark grey boxes for Foundation and Economy memberships, a golden small box with a link to a modal, and a golden footer with a WhatsApp icon. The modal contains a simple contact form with email and message fields.

Custom CSS styles define the dark grey background for content boxes, golden yellow for headers and accents, light blue for the Economy header, soft pastels for member boxes, and white text on dark backgrounds for readability. Colors were chosen based on psychological meanings: dark grey for stability, golden for prestige and attention, light blue for trust, pastels for friendliness and harmony. Techniques include complementary and analogous color schemes, a neutral background with accent colors, consistent branding, and high contrast for accessibility.

JavaScript provides interactivity: a menu toggle that shows a dropdown, toggling of membership descriptions when clicking on titles.

The design is fully responsive using Bootstrap's grid system, adapting to desktop, tablet, and mobile views.
